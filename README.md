# miniAapps
小程序
功能：

1. 登录输入账号密码动画
2. 自动选定和缓存工作地点，以及选中工作任务。
3. 长按圆圈倒计时打卡
4. 摇一摇下班打卡
5. 工作日志默认“下班打卡”
